    My media server

    I have setup a media server to watch some contents at home. Also available from external network trough a VPN.

    Hardware

    My media server run on a NiPoGi AK2 PLUS Mini PC with following specifications :

    Processor   :   Intel Alder Lake-N100 
RAM         :   16GB DDR4
Storage     :   512GB M.2 SSD

    I also use a Raspberry Pi 3 B+ who host the VPN and a DNS server.

    Software

    I use different application to run my media server. All these applications run trough Docker container.

    Tranmission-OpenVPN

    I use Transmission-OpenVPN to download my favorite Linux ISO trough torrent.

    This is a Docker container running Transmission torrent client with WebUI over an OpenVPN tunnel
    I use a VPN to avoid exposing my public IP on Internet, I use PIA as VPN provider.

    Jackett

    I use Jackett as indexer

    Radarr

    Sonarrr

    Jellyfin

    Jellyfin is a Free Software Media System that permit you to read your different media.
    I use it trough a Docker container made by LinuxServer.io.

    Backup

    I backup my data every day (exept the ISO files because is too large).
    To made the backup I use 2 basic bash scripts :

    backup.sh

    #!/bin/bash

# Chemin du dossier de sauvegarde
backup_folder="$HOME/backup"

# Chemin du dossier de travail
workspace_folder="$HOME/workspace"

# Chemin du fichier de log
log_file="$workspace_folder/scripts/logs/backup_script.log"

# Créer le dossier de sauvegarde s'il n'existe pas
if ! mkdir -p "$backup_folder"; then
    echo "$(date +'%Y-%m-%d %H:%M:%S') - Erreur lors de la création du dossier de sauvegarde" >> "$log_file"
    exit 1
fi

# Obtenir l'heure précise au format HH-MM-SS
current_time=$(date +'%H-%M-%S')

# Liste des fichiers à sauvegarder
files_to_backup=(
    "cocoflix/docker-compose.yml"
    "cocoflix/config/jackett"
    "cocoflix/config/proxy"
    "cocoflix/config/transmission/transmission-home"
    "cocoflix/config/radarr/Backups"
    "cocoflix/config/sonarr/Backups"
)

# Exécuter la commande tar avec le chemin dynamique et rediriger la sortie vers le fichier de journal
tar_output=$(tar -czvf "$backup_folder/archive_$(date +'%d-%m-%Y')_$current_time.tar.gz" -C "$workspace_folder" "${files_to_backup[@]}" 2>&1)

# Délimiteur
delimiter="==============================="

# Vérifier le code de retour de la commande tar
if [ $? -eq 0 ]; then
    # Journaliser le succès de la sauvegarde
    echo -e "$(date +'%Y-%m-%d %H:%M:%S') - Backup réussi\n$delimiter\nDétails :\n$tar_output\n$delimiter" >> "$log_file"
else
    # Journaliser l'erreur en cas d'échec de la commande tar
    echo -e "$(date +'%Y-%m-%d %H:%M:%S') - Erreur lors de la sauvegarde\n$delimiter\nDétails :\n$tar_output\n$delimiter" >> "$log_file"
fi

    And as I want to keep only 7 backups, I use following scripts to clean the old files.

    clean_backup.sh

    #!/bin/bash

backup_dir="$HOME/backup"
keep_backups=7
log_file="$HOME/workspace/scripts/logs/backup_clean_script.log"
backup_pattern="archive_"

# Fonction pour journaliser les messages
log() {
    echo "$(date +"%Y-%m-%d %H:%M:%S"): $1" >> "$log_file"
}

# Vérification de l'existence du répertoire de sauvegarde
if [ ! -d "$backup_dir" ]; then
    log "Le répertoire de sauvegarde $backup_dir n'existe pas. Arrêt du script."
    exit 1
fi

# Vérification des autorisations d'écriture
if [ ! -w "$backup_dir" ]; then
    log "Le script n'a pas les autorisations d'écriture dans le répertoire de sauvegarde $backup_dir. Arrêt du script."
    exit 1
fi

cd "$backup_dir" || exit

log "---- Début du script ----"

# Compte le nombre total de backups
total_backups=$(find . -maxdepth 1 -type f -name "$backup_pattern*" | wc -l)
log "Nombre total de sauvegardes: $total_backups"

# Supprime les backups les plus anciens excédant le nombre souhaité
if [ "$total_backups" -gt "$keep_backups" ]; then
    backups_to_remove=$((total_backups - keep_backups))
    log "Nombre de sauvegardes à supprimer: $backups_to_remove"

    # Liste les fichiers à supprimer
    files_to_remove=$(find . -maxdepth 1 -type f -name "$backup_pattern*" | sort -r | tail -n "$backups_to_remove")

    # Supprime les fichiers
    echo "$files_to_remove" | xargs rm -f

    # Vérification des erreurs lors de la suppression
    if [ $? -eq 0 ]; then
        log "Fichiers supprimés avec succès:"
        log "$files_to_remove"
    else
        log "Erreur lors de la suppression des fichiers. Veuillez vérifier les autorisations ou les fichiers verrouillés."
    fi
fi

log "---- Fin du script ----"
